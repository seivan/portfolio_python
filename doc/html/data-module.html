retrieve_projects(sort_by='start_date', sort_order='asc', techniques=None, search=None, search_fields=None)

source code 

name: retrieve_projects

parameter: sort_by(string): sorting depending on the keyvalue from projects, default is 'start_date'

parameter: sort_order(string): the sort order, normal if asc, reversed if desc, default is asc.

parameter: techniques(list): searching techniques within each project, default is None

parameter: search(string): free text search, searches through projects, default is None

parameter: search_fields(list): defines what key keyvalue from projects to search through

description: This function search for projects depending on what you search for and which search_fields you define to search on. The user can filter which project to show depending on which techniques they used. The output can be choosed to be sorted by user field-wise (sort_by) and either in falling or reverse order.
